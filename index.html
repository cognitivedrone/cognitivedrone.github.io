Cognitive Drone: A VLA Model and Evaluation Benchmark for Real-Time Cognitive Task Solving and Reasoning in UAVs

Welcome to the official project page for the Cognitive Drone dataset and benchmark. This project focuses on advancing real-time cognitive task solving and reasoning in Unmanned Aerial Vehicles (UAVs) using Vision-Language-Action (VLA) models.

Authors: Artem Lykov, Valerii Serpiva, Muhammad Haris Khan, Oleg Sautenkov, Artyom Myshlyaev, Grik Tadevosyan, Yasheerah Yaqoot, and Dzmitry Tsetserukou
